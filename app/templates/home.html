AudioRama is an audiobook recommendation application designed and developed for Audible listeners and subscribers. Current Audible users get a more holistic set of recommendations built using a wide variety of information. This includes information ranging from a user's profile to current popular books on the website. However, if you are particularly interested in discovering new narrators, you would have to look through your list of 50-odd recommendations to find a new narrator you might enjoy.

If you are a listener who listens to books purely for the narration experience; and is interested in finding new phenomenal narrators in genres that you are interested in, AudioRama is for you. AudioRama will help you discover new books by discovering new narrators. This is a tool to supplement to Audible's recommendations.

To use the application, navigate to the login page, to learn more about the underlying model and benchmarks, visit the about page. You can also explore AudioRama's database, and Audible's user profile.
